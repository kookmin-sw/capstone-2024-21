This package includes 3 Lightning effects.
(Mesh, Thunder and LightningMagic)

----------------------------
Quick Start
----------------------------
Open "Demo.unity"
(Assets/LightningPack/Demo/Demo.unity)
hit Play button, You can see the particles!
hit "SpaceKey",play Thunder partilce in PlayMode.

----------------------------
Change the Color
----------------------------
in the Demo scene, You can change particle color.
Check the "Demo" in hierarchy, attached "Demo" script.
Set the particle color and hit "SpaceKey", You can change all particle color!

----------------------------
How to use
----------------------------
Drop prefabs into your scene.
Set the Collision, if particle parameter check is ON.

!!!!All Particle Play Collect, if you do not set Collision!!!!
